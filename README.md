# Profilr: AI-Powered Career Management Hub

Profilr is a comprehensive career management platform designed for students and professional job-seekers to optimize their applications, track their progress, and showcase their professional identity. By leveraging state-of-the-art NLP and generative AI, Profilr bridge the gap between candidate qualifications and recruiter requirements.

## Key Features

*   **ATS Score Analysis**: Comprehensive assessment of resumes using NLP-based analysis to determine compatibility with industry standards and specific job descriptions.
*   **AI Recommendations**: Targeted, high-impact suggestions generated by Google Gemini 2.0 Flash and Groq (Llama-3) to refine resume content and overcome common ATS filters.
*   **Placement Kanban Dashboard**: A centralized management interface to track job applications through various stages including Applied, Interviewing, Technical Round, Offer, and Rejected.
*   **LinkedIn Integration**: Automated extraction and importation of LinkedIn profile data via Apify to streamline the creation of professional portfolios.
*   **Dynamic Public Portfolios**: Generation of high-end, customizable public portfolios that allow candidates to present a cohesive professional brand to potential employers.
*   **Premium Subscription Tier**: Enhanced features available through a tiered subscription model, managed via a secure payment gateway.

## Tech Stack

### Frontend
- **React + Vite**: High-performance single-page application framework.
- **Glassmorphism UI**: Advanced CSS aesthetics for a premium, modern user experience.
- **Tailwind CSS**: Utility-first styling for responsive and consistent layouts.
- **Lucide React**: Vector-based iconography for a clean visual interface.

### Backend
- **FastAPI**: Asynchronous Python framework for high-concurrency API performance.
- **Supabase**: Backend-as-a-Service providing PostgreSQL database, user authentication, and real-time data synchronization.
- **NLP (spaCy)**: Deep text processing and entity recognition for calculating ATS compliance scores.
- **Multi-Model AI Orchestration**: Integration with Google Gemini and Groq for generating contextual career advice and resume improvements.
- **Razorpay Integration**: Secure payment processing for subscription management.

## Payments and Subscription

Profilr implements a Pro subscription model to unlock advanced analytics and premium portfolio templates.

### Razorpay Sandbox
The platform currently utilizes the Razorpay Sandbox environment for payment simulation. This allows users to experience the full upgrade flow without actual financial transactions.
- **Testing Credentials**: In the testing phase, any valid UPI ID or test card details provided by Razorpay can be used to simulate a successful payment.
- **Automatic Provisioning**: Upon successful payment verification via HMAC-SHA256 signature checking, the backend automatically promotes the user account to the "Pro" tier in the Supabase database.

## Infrastructure and Setup

### Prerequisites
- Python 3.10+
- Node.js 18+
- Supabase Account
- API Keys for Gemini, Groq, and Apify

### Backend Installation
1. Initialize a virtual environment:
   ```bash
   python -m venv spd_venv
   source spd_venv/bin/activate
   ```
2. Install the required Python packages:
   ```bash
   pip install -r backend/requirements.txt
   ```
3. Configure environment variables in a `.env` file within the backend directory:
   ```text
   GOOGLE_API_KEY=your_key
   GROQ_API_KEY=your_key
   APIFY_API_KEY=your_key
   SUPABASE_URL=your_url
   SUPABASE_KEY=your_key
   RAZORPAY_KEY_ID=your_id
   RAZORPAY_KEY_SECRET=your_secret
   ```
4. Launch the FastAPI server:
   ```bash
   python backend/main.py
   ```

### Frontend Installation
1. Navigate to the frontend directory and install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Create a `.env` file for the frontend:
   ```text
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key
   ```
3. Execute the development build:
   ```bash
   npm run dev
   ```
